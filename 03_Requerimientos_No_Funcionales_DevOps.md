# Requerimientos No Funcionales DevOps

## Definición
Son características relacionadas con calidad, rendimiento, seguridad y operación del sistema.

## CI/CD
- RNF-DEV-01: Pipeline automatizado
- Métrica: Deploy en < 10 minutos

## Control de Versiones
- RNF-DEV-02: Uso de Git
- Métrica: 100% del código versionado

## Pruebas
- RNF-DEV-03: Pruebas automáticas
- Métrica: Cobertura ≥ 80%

## Entornos
- RNF-DEV-04: Dev, QA y Producción separados

## Contenerización
- RNF-DEV-05: Uso de Docker

## Monitoreo
- RNF-DEV-06: Monitoreo en tiempo real
- Alertas en < 1 minuto

## Logging
- RNF-DEV-07: Logs centralizados

## Seguridad
- RNF-DEV-08: Gestión segura de credenciales

## Vulnerabilidades
- RNF-DEV-09: Escaneo automático

## Backups
- RNF-DEV-10: Backups diarios

## Recuperación
- RNF-DEV-11: DRP
- Tiempo de recuperación < 1 hora

## Infraestructura
- RNF-DEV-12: Infraestructura como código (IaC)
